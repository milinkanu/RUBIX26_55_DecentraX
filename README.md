1.  **Post**: User posts a **Lost** item (asking for help) or a **Found** item (offering help).
2.  **Verify**: If it's a "Found" item, the finder sets a secret question.

5.  **Connect**: If verified, a secure chat channel opens up.
6.  **Recover**: Users meet safely, exchange the item, and mark the case as **Resolved**.
